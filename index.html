1. Introduction

Bienvenue sur Houda Al Fourquan. Cette politique de confidentialité explique comment nous traitons vos données lorsque vous utilisez notre application islamique.

مرحبًا بكم في هدى الفرقان. تشرح سياسة الخصوصية هذه كيفية تعاملنا مع بياناتك عند استخدامك لتطبيقنا الإسلامي.

Notre engagement : Votre vie privée est notre priorité. Nous avons conçu cette application pour fonctionner avec un minimum de données et un maximum de confidentialité.

التزامنا: خصوصيتك هي أولويتنا. لقد صممنا هذا التطبيق للعمل بأقل قدر من البيانات مع أقصى قدر من الخصوصية.

2. Données Collectées

❌ Données NON collectées

Nous ne collectons AUCUNE des données suivantes :

  • ❌ Nom et prénom / الاسم واللقب
  • ❌ Adresse email / البريد الإلكتروني
  • ❌ Numéro de téléphone / رقم الهاتف
  • ❌ Adresse postale / العنوان البريدي
  • ❌ Historique de navigation / سجل التصفح
  • ❌ Identifiants de publicité / معرفات الإعلانات
  • ❌ Données de paiement (aucun paiement requis) / بيانات الدفع (لا توجد مدفوعات مطلوبة)

✅ Données utilisées localement

Les données suivantes sont utilisées UNIQUEMENT sur votre appareil et ne sont JAMAIS transmises :

  • Position GPS : Utilisée uniquement pour calculer les horaires de prière et la direction Qibla. / الموقع الجغرافي: يستخدم فقط لحساب أوقات الصلاة واتجاه القبلة.
  • Préférences utilisateur : Vos paramètres sont stockés localement. / تفضيلات المستخدم: تُخزن إعداداتك محليًا على جهازك.
  • Historique de lecture : Votre progression dans le Coran et les tasbihs est conservée uniquement sur votre appareil. / سجل القراءة: يتم الاحتفاظ بتقدمك في القرآن والأذكار على جهازك فقط.
  • Compteurs Tasbih : Vos comptes de dhikr sont stockés localement. / عدادات التسبيح: يتم تخزينها محليًا على جهازك.

3. Permissions Utilisées

📍 Localisation (GPS)

Pourquoi ? : Calcul précis des horaires de prière et de la direction Qibla. / لماذا؟: لحساب أوقات الصلاة واتجاه القبلة بدقة.

Comment ? : La position est utilisée uniquement pendant l'utilisation de l'application. / كيف؟: الموقع يُستخدم فقط أثناء استخدام التطبيق.

Type : "Toujours" ou "Lors de l'utilisation" selon votre choix. / النوع: "دائمًا" أو "أثناء الاستخدام" حسب اختيارك.

🌐 Internet

Pourquoi ? : Télécharger les récitations audio du Coran et les traductions. / لماذا؟: لتنزيل تلاوات القرآن وترجماته.

Comment ? : Connexion uniquement lors du téléchargement de contenu. / كيف؟: الاتصال فقط عند تنزيل المحتوى.

🔔 Notifications

Pourquoi ? : Rappels pour les prières et l'Adhan. / لماذا؟: للتنبيهات الخاصة بأوقات الصلاة والأذان.

Comment ? : Notifications locales programmées sur votre appareil. / كيف؟: إشعارات محلية مبرمجة على جهازك.
