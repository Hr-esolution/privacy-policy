Politique de Confidentialité

📅 Dernière mise à jour : Mars 2026

✅ En résumé : Cette application ne collecte AUCUNE donnée personnelle.

1. Introduction

Bienvenue sur Houa Al Fourquan. Cette politique de confidentialité explique comment nous traitons vos données lorsque vous utilisez notre application islamique.

Notre engagement : Votre vie privée est notre priorité. Nous avons conçu cette application pour fonctionner avec un minimum de données et un maximum de confidentialité.

2. Données Collectées

❌ Données NON collectées

Nous ne collectons AUCUNE des données suivantes :

  • ❌ Nom et prénom
  • ❌ Adresse email
  • ❌ Numéro de téléphone
  • ❌ Adresse postale
  • ❌ Historique de navigation
  • ❌ Identifiants de publicité
  • ❌ Données de paiement (aucun paiement requis)

✅ Données utilisées localement

Les données suivantes sont utilisées UNIQUEMENT sur votre appareil et ne sont JAMAIS transmises :

  • Position GPS : Utilisée uniquement pour calculer les horaires de prière et la direction Qibla. La position n'est pas stockée et n'est jamais envoyée à nos serveurs ou à des tiers.
  • Préférences utilisateur : Vos paramètres (méthode de calcul, langue, notifications) sont stockés localement sur votre appareil.
  • Historique de lecture : Votre progression dans le Coran et les tasbihs est conservée uniquement sur votre appareil.
  • Compteurs Tasbih : Vos comptes de dhikr sont stockés localement.

3. Permissions Utilisées

📍 Localisation (GPS)

Pourquoi ? : Calcul précis des horaires de prière et de la direction Qibla.

Comment ? : La position est utilisée uniquement pendant l'utilisation de l'application pour les calculs. Elle n'est pas stockée.

Type : "Toujours" ou "Lors de l'utilisation" selon votre choix.

🌐 Internet

Pourquoi ? : Télécharger les récitations audio du Coran et les traductions.

Comment ? : Connexion uniquement lors du téléchargement de contenu. Aucune donnée n'est envoyée.

🔔 Notifications

Pourquoi ? : Rappels pour les prières et l'Adhan.

Comment ? : Notifications locales programmées sur votre appareil.

4. Partage de Données

✅ Nous ne vendons, ne louons et ne partageons AUCUNE donnée avec des tiers.

Il n'y a :

  • ❌ Pas de publicité (donc pas de partage avec régies publicitaires)
  • ❌ Pas de analytics (donc pas de partage avec Google Analytics, Firebase, etc.)
  • ❌ Pas de réseaux sociaux intégrés (donc pas de partage avec Facebook, Twitter, etc.)
  • ❌ Pas de vente de données

5. Services Tiers

L'application utilise les services suivants pour fournir du contenu :

  • Quran.com API : Pour le texte du Coran et les traductions. Aucune donnée personnelle n'est envoyée.
  • EveryAyah.com : Pour les récitations audio. Les fichiers sont téléchargés sans transmission de données personnelles.
  • OpenStreetMap Nominatim : Pour la géolocalisation des villes. Seules les coordonnées GPS sont envoyées temporairement.

Important : Ces services reçoivent uniquement des coordonnées GPS anonymes pour fournir le contenu demandé. Aucun identifiant personnel n'est transmis.

6. Stockage des Données

Où sont stockées vos données ?

Toutes vos données sont stockées localement sur votre appareil (iPhone/iPad ou Android).

Combien de temps ?

Les données sont conservées tant que l'application est installée. Si vous désinstallez l'application, toutes les données sont supprimées automatiquement.

7. Sécurité

Bien que nous ne collections pas de données personnelles, nous prenons la sécurité au sérieux :

  • ✅ Les données locales sont stockées dans le bac à sable (sandbox) de l'application
  • ✅ Aucune transmission réseau de données personnelles
  • ✅ Connexions HTTPS sécurisées pour le téléchargement de contenu

8. Droits des Utilisateurs

Conformément au RGPD (Europe) et autres réglementations, vous avez les droits suivants :

  • Droit d'accès : Vous pouvez voir toutes vos données dans l'application
  • Droit à la suppression : Désinstallez l'application pour supprimer toutes vos données
  • Droit à la portabilité : Non applicable (pas de données collectées)
  • Droit d'opposition : Vous pouvez désactiver la localisation dans les paramètres de votre appareil

9. Enfants et Vie Privée

Notre application :

  • ✅ Ne collecte aucune donnée personnelle
  • ✅ Ne cible pas spécifiquement les enfants de moins de 13 ans
  • ✅ Est conforme avec la loi COPPA (Children's Online Privacy Protection Act)

Les enfants peuvent utiliser l'application en toute sécurité car aucune donnée n'est collectée ou transmise.
